  • Color in CSS

    1. Hexadecimal(16 number):


      # + String of 6 hexadecimal numbers(from 0 - F)


      first two digits –> how much red;


      next two digits –> how much green;


      last two digits –> how much blue;


      Generally using color picker

    2. RGB:


      3 channels: Red, Green, Blue. Each ranges from 0-255


      color: rgb(0, 255, 0);

    3. RGBA:


      just like RGB, but with an alpha(transparency) channel from 0.0 - 1.0(lower means sligter)


      color: rgba(11, 99, 150, 1);

  • CSS Selector

    • Element Selector: Select all instances of a given element
