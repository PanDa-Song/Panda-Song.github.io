1-JavaScript

Fundation

  • Primitives

    • Numbers
    • Strings
      • “hello”.length
      • “hello”[0]
    • Booleans
    • null & undedined

  • Variables

    • A kind of contaner that store data
    • Syntax:
      var yourVariableName = yourValue

  • Methods

    • Alert
    • Prompt
    • Console.log

Control Flow

  • Boolean

    • Comparison Operator
      • === vs. ==
    • Logical Operator
      • &&, ||, !
      • Every values are inherently “truely” or “falsey” when evaluate in a boolean context
      • false, 0, “”, null, undefined, NaN — false

  • Conditionals

    • if
    • while
    • for

Function

  • Basic syntax is similar to Java.
  • Higher order functions
    • setInterval(function, interval(in ms)) / clearInterval

Arrays

  • Basic
    • Two methods to define:
      • var x = [];
      • var x = new Array();
    • Can store any type of data together
  • Method
    • push / pop
    • shift / unshift
    • indexOf
    • slice
    • splice (= delete)
  • Iteration
    • for loop
    • forEach (common)

Object

  • Just like HashMap in Java or dictionary in Python.
  • Syntax is the same as dictionary in Python

2-DOM

Introduction

  • Document Object Model –> Interface between JavaScript and HTML+CSS
  • Every elements in HTML are transfered as objects in JavaScript
  • document.someElement

DOM Selector

  • Five Methods
    • document.getElementById()
    • document.getElementsByClassName()
    • document.getElementsByTagName()
    • document.querySelector()
    • document.querySelectorAll()

DOM Manipulation

  • change an element’s style
    • property like CSS
    • tag.style.color/border.etc
  • adding / removing classes
    • tag.classList.add/remove/toggle
  • changing the contents of a tag
    • tag.textContent
    • tag.innerHTML //reserve the html tag in the content
  • changing attrobutes(src, href, etc)
    • var.getAttribute
    • var.setAttribute

Events

  • Making the webpage interactive
  • Examples:
    • clicking on a button
    • hovering over a link
    • dragging and droping
    • pressing the enter key
  • Process
    • select an element
    • add an event listener
  • Syntax
    • element.addEventListener(type, functionToCall);
    • eg. 
      var button = document.querySelector("button");
button.addEventListener("click", function(){
 console.log("Someone clicked the button");
});

3-JQuery

  • JQuery is a DOM manipulation library
  • Opponion on JQuery
    • USE
      • Fixes “broken” DOM API
      • Brievity and Clarity
      • Ease of use
      • Cross-Browser Support
      • AJAX
      • Lot’s of people use it
    • NOT USE #Reference
      • The DOM API is no longer “broken”
      • It doesn’t do anything you can do on your own
      • It’s an unnecessary dependency
      • Performance
      • Lot’s of people are moving away from JQuery

  • JQuery Selector

    • similar to querySelectorALL
    • Syntax:
      $("selectorName")
    • Manipulating Style Using CSS
      $("selector).css("property", "value")

  • Methods

    • Basic

      • text()
      • html()
      • attr()
      • val()
      • addClass()
      • removeClass()
      • toggleClass()

    • Events

      • click()
      • keyPress()
      • on()

    • Effects 不同的特效

      • fadeOut()
      • fadeIn()
      • fadeToggle()
      • slideUp()
      • slideDown()
      • slideToggle()
