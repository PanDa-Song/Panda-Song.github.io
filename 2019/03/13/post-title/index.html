windows7下用Hexo+github建立博客

安装Hexo

本地电脑中需要提前安装两个需要的程序
-Node.js
-Git

Git

windows电脑从网站上下载安装程序安装到电脑上，由于国内网络的原因，用官网的源下载速度会很慢，因此选择国内的淘宝的镜像下载git。下载好后打开exe文件根据提示进行安装。

Node.js

与git一样，下载安装程序到电脑（可以在官网node.js）,与普通的应用程序一样安装即可。

两者安装好后，打开powershell或cmd确认系统是否已经安装成功。

  • node -v
  • git version
    安装好后进行下一步，安装Hexo。

安装Hexo

npm install -g hexo-cli
安装好后即可开始建站。

建站

hexo init Hexo
创建名为Hexo的文件夹并存放Hexo建站相关文件
cd Hexo # 进入刚刚创建的文件夹

开始建站：
hexo generate
生成静态文件
hexo server
启动服务器，使网页在本地生成，默认情况下，地址为：http://localhost:4000/ ，可在本地浏览器中查看。

现在网站已经建立起来，接来下需要借助github使网站能够从Internet访问。

配置Github与Git

  • 打开Git Bash

  • 配置name和email
    输入git config --global user.name"yourname"设置Git的用户名
    输入git config --global user.email"youremail"设置Git的邮箱

  • 生成秘钥

在用户目录下创建.ssh文件夹添加秘钥
ssh-keygen -t rsa -C "yourEmail"
连续三个回车后，生成秘钥，即.ssh文件夹中的’id_rsa.pub’文件

  • 创建github帐号

  • 添加秘钥到github仓库
    点击右上角的加号-new repository，repository name即为’yourname.github.io’，其他默认选项即可。
    点击加号旁边另一个向下箭头，选择setting-SSH and GPG keys，在SSH keys中新建key，将刚刚的’id_rsa.pub’文件中内容复制过来。
    Git Bash中输入ssh -T git@github.com，测试秘钥添加是否成功，若Hi之后为用户名则说明添加成功。

将hexo和github关联

进入之前建站创建的Hexo文件夹，找到_config.yml文件（最关键的文件），用编辑器打开进行修改，在末尾进行如下修改。
() deploy: type: git repository: git@github.com:yourname/yourname.github.io.git branch: master ()

接下来进行博客的部署，用cmd或者Git Bash进入创建博客的文件夹即Hexo，生成并部署到github。
hexo g
hexo d

大功告成！

此时在浏览器输入https://yourname.github.io/即可访问个人博客。
