算法入门

这篇文章打算介绍并总结一下一些基础的数据结构和算法以及他们的实现以及例子。给自己后面的刷题以及更高级算法的学习提供帮助。

数据结构

数组 Array

  • 用于存储固定大小的同类型的的元素
  • 在内存中，占据一块连续的区域。因此需要在创建时预留空间，例如int[] arr = int[10],因此可能会导致内存空间的浪费
  • 由于数组中元素的存储地址是连续的，所以可以通过下标的方式直接查找以及修改，时间复杂度为O(1)**。但是对于插入或这删除元素，花费则取决于插入或删除的位置，如插入的位置在最后一位则花费O(1)时间，但如果插入在第一位，则插入时后面所有的元素需要向后移一位，时间复杂度为O(N)**
  • 在Java集合中的实现是ArrayList

链表 LinkedList

  • 与数组一样属于线性表，但是在每个节点除了存储相应的元素以外，还包括了指向下一个节点的地址
  • 在内存中，可以分散存储在任何位置。不需要预先指定大小，便于节省内存空间
  • 与数组相反，由于存储结构并不连续无法通过下标直接定位元素，查找时需要从头节点开始一个一个向后找，时间复杂度为O(N)**，而插入或者删除新元素则只需要改变前趋节点的指针指向的地址，也就是O(1)**时间复杂度
  • 相较于普通的链表，还存在其他用于实现不同功能的链表，
  • 在Java集合中的实现是LinkedList

栈 Stack

  • 一种特殊的线性表，插入和删除操作只能在栈顶也就是表的末尾进行，也就是只能针对最后加入的元素进行操作(LIFO)
  • 数据模型就像是一摞书，只有最上面的一本书是可见的，可以对他进行操作
