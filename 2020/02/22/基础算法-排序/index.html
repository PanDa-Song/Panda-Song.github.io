排序算法

Selection Sort

算法：
选择数组中最小的元素放到最前面，再从剩下的元素中选择最小的元素放到他的后面，重复此过程直到只剩一个元素。

复杂度：
每次选择最小元素时需要遍历所有剩下的元素，然后将其交换到相应的位置，因此需要N次交换，(N-1)+(N-2)…+2+1+0 = N(N-1)/2次比较，即O(N(N+1)/2)=O(N^2)

实现：

public void selectionSort(int[] arr) {
    for (int i = 0; i < arr.length; i++) {
        int min = i;
        for (int j = i + 1; j < arr.length; j++) {
            if (arr[i] > arr[j]){
                min = j;
            }
        }
        int temp = arr[i];
        arr[i] = arr[min];
        arr[min] = temp;
    }
}

Insertion Sort

算法：
通过插入以及后移使数组左边为有序序列右边为无序序列，重复该过程，使右边的无序序列为0。从index=1的元素开始向后，跟左边的有序序列进行逐一比较（从index-1开始从右往左），当该元素大于有序序列的第i个元素，将该元素插入到i+1的位置。

复杂度：
最差情况N(N-1)/2次比较，N(N-1)/2次交换，最好情况只有N-1次比较无需交换，时间复杂度O(N^2)

实现：

public void insertionSort(int[] arr) {
    for (int i = 1; i < arr.length; i++) {
        int insertVal = arr[i];
        int index = i - 1;

        while (index != 0 && insertVal < arr[index]) {
            arr[index + 1] = arr[index];
            index --;
        }
        arr[index + 1] = insertVal;
    }
}

Shellsort

算法：
基于插入排序的一种算法。首先对数组进行分组，选择N/2作为分组的gap，也就是对pointer1=0与pointer2=gap的数据进行大小比较，若顺序不对，则交换他们，若顺序正确，则不交换，无论他俩顺序如何，都需要向前继续以gap的长度作为间隔进行比较，完成该步骤后，两个指针均向后移动一位，直到pointer2到达数组结尾。完成这一遍排序后，gap继续除2，重复上述步骤，直至gap=1为最后一遍排序，得到排好序的数组。

复杂度：
最差情况O(N^2)，平均时间只有O(N*logN)。尽管时间复杂度也为O(N^2)，但是希尔排序是按照不同间隔对元素进行插入排序，当刚开始元素很无序的时候，间隔最大，插入排序的元素个数很少，速度很快；当元素基本有序了，步长很小，插入排序对于有序的序列效率很高。所以，希尔排序的时间复杂度会比O(N^2)好一些。

实现：

public int[] ShellSort(int[] nums) {
    int n = nums.length;
    for (int gap = n / 2; gap >= 1; gap /= 2) {
        // j -- 指针 2
        for (int j = gap; j < n; j++) {
            // i -- 指针 1
            for (int i = j - gap; i >= 0; i -= gap) {
                if (nums[i + gap] > nums[i]) {
                    break;
                } else {
                    int temp = nums[i];
                    nums[i] = nums[i + gap];
                    nums[i + gap] = temp;
                }
            }
        }
    }
    return nums;
}

Quick Sort

public static int[] qsort(int arr[],int start,int end) {        
    int pivot = arr[start];        
    int i = start;        
    int j = end;        
    while (i<j) {            
        while ((i<j)&&(arr[j]>pivot)) {                
            j--;            
        }            
        while ((i<j)&&(arr[i]<pivot)) {                
            i++;            
        }            
        if ((arr[i]==arr[j])&&(i<j)) {                
            i++;            
        } else {                
            int temp = arr[i];                
            arr[i] = arr[j];                
            arr[j] = temp;            
        }        
    }        
    if (i-1>start) arr=qsort(arr,start,i-1);        
    if (j+1<end) arr=qsort(arr,j+1,end);        
    return (arr);    
}
