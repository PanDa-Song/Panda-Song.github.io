windows7下用Hexo+github建立博客

安装Hexo

本地电脑中需要提前安装两个需要的程序
-Node.js
-Git

Git

windows电脑从网站上下载安装程序安装到电脑上，由于国内网络的原因，用官网的源下载速度会很慢，因此选择国内的淘宝的镜像下载git。下载好后打开exe文件根据提示进行安装。

Node.js

与git一样，下载安装程序到电脑（可以在官网node.js）,与普通的应用程序一样安装即可。

两者安装好后，打开powershell或cmd确认系统是否已经安装成功。

  • node -v
  • git version
    安装好后进行下一步，安装Hexo。

安装Hexo

npm install -g hexo-cli
安装好后即可开始建站。

建站

hexo init Hexo
创建名为Hexo的文件夹并存放Hexo建站相关文件
cd Hexo # 进入刚刚创建的文件夹

开始建站：
hexo generate
生成静态文件
hexo server
启动服务器，使网页在本地生成，默认情况下，地址为：http://localhost:4000/ ，可在本地浏览器中查看。

现在网站已经建立起来，接来下需要借助github使网站能够真的登录上网。
